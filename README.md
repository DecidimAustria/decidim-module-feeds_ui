# Decidim::FeedsUi

A mobile ui for the decidim feeds module.

## Usage

FeedsUi will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-feeds_ui"
```

And then execute:

```bash
bundle
```
